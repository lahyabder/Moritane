# Moritane Enhancement Plan

This document outlines the roadmap for upgrading the Moritane streaming application.

## 1. Core Features
- [ ] **Video Player**: Implement a custom video player overlay/modal with playback controls (mocked for now).
- [ ] **Search Functionality**: Add a real-time search bar in the Navbar to filter movies and series.
- [ ] **Authentication**: Add a mock Login/Register flow with user profiles.

## 2. Design & UX Improvements
- [ ] **Animations**: Add smooth entry animations, hover effects on cards, and page transitions.
- [ ] **Responsive Design**: Ensure mobile-friendliness for the new features.
- [ ] **Loading States**: Add skeleton loaders for a premium feel.

## 3. Content Expansion
- [ ] **Mock Data**: Add more categories (Documentaries, Kids, Music).
- [ ] **Detailed Info**: Add "More details" data like cast, duration, and genres.

## Current Focus
We will start by implementing the **Video Player** and **Search Functionality**.
